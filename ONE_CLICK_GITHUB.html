🚀 Publish Kopitiam Website to GitHub

✅ Your website is ready! Follow these 3 simple steps to publish it on GitHub.

1 Create GitHub Repository

Click the button below to create a new repository on GitHub:

Create GitHub Repository
Important: When creating the repository:
  • Repository name: kopitiam-website (or your preferred name)
  • Description: Website for Kopitiam Rasa
  • Visibility: Public
  • Initialize with README: UNCHECK THIS

2 Run These Commands

Open Terminal/Command Prompt and run:

# Navigate to website folder
cd /data/.openclaw/workspace/kopitiam-website

# Add your GitHub repository (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/kopitiam-website.git

# Push to GitHub
git push -u origin main

Note: When prompted for password, use your GitHub password. If you have 2FA enabled, use a Personal Access Token.

3 Enable GitHub Pages

Click the button below to enable GitHub Pages:

Enable GitHub Pages

Then configure:

  • Source: Deploy from a branch
  • Branch: main
  • Folder: / (root)
  • Click Save

🎉 Your Website Will Be Live At:

https://YOUR_USERNAME.github.io/kopitiam-website/

Wait 1-2 minutes after saving GitHub Pages settings.

⚠️ Troubleshooting

If git push fails:

  • Make sure you created the repository first
  • Check your username is correct
  • Use Personal Access Token if you have 2FA enabled
  • Try SSH method: git remote set-url origin git@github.com:YOUR_USERNAME/kopitiam-website.git
