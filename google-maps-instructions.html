Google Maps Embed for Kopitiam Rasa

Address to Use:

18 Jalan Klang Sentral 2/KU5, Klang, Selangor, Malaysia

How to Add Google Maps to Your Website:

Step 1: Get Google Maps Embed Code

  1. Go to Google Maps
  2. Search for: 18 Jalan Klang Sentral 2/KU5, Klang, Selangor
  3. Click on the location pin
  4. Click the "Share" button
  5. Select the "Embed a map" tab
  6. Choose your preferred map size (recommended: Medium 600x450)
  7. Copy the embed code

Step 2: Replace the Map Placeholder

In index.html, find this section (around line 280-290):

<div class="map-placeholder">
    <h4>Map Location</h4>
    <p>Klang, Selangor, Malaysia</p>
    <div class="map-image">
        <!-- In a real website, you would embed Google Maps here -->
        <img src="https://images.unsplash.com/..." alt="Map Location">
    </div>
</div>

Replace it with:

<div class="map-placeholder">
    <h4>Our Location</h4>
    <p>18 Jalan Klang Sentral 2/KU5, Klang</p>
    <div class="map-embed">
        <!-- Paste your Google Maps embed code here -->
        <iframe 
            src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3983.534123456789!2d101.4456789!3d3.0456789!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x31cc4d1234567890%3A0xabcdef123456789!2s18%20Jalan%20Klang%20Sentral%202%2FKU5%2C%20Klang%2C%20Selangor!5e0!3m2!1sen!2smy!4v1234567890123!5m2!1sen!2smy" 
            width="600" 
            height="450" 
            style="border:0;" 
            allowfullscreen="" 
            loading="lazy" 
            referrerpolicy="no-referrer-when-downgrade">
        </iframe>
    </div>
</div>

Step 3: Update CSS (Optional)

Add this to your css/style.css to style the iframe:

.map-embed iframe {
    width: 100%;
    height: 300px;
    border-radius: 10px;
    border: none;
}
