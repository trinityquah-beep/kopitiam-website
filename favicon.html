Suggested design: Brown background with "KR" in white

How to create a favicon:

  1. Create a 64x64 pixel image with a brown (#8B4513) background
  2. Add white text "KR" (for Kopitiam Rasa) in the center
  3. Save it as favicon.ico in the website root folder
  4. Also create PNG versions (16x16, 32x32, 64x64, 128x128, 256x256)
  5. Update the HTML to include all sizes in the <head> section

Quick online tools:

  • Favicon Generator
  • Favicon.io
  • Real Favicon Generator

Recommended HTML for favicon (add to <head>):

<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="manifest" href="/site.webmanifest">
<link rel="mask-icon" href="/safari-pinned-tab.svg" color="#8b4513">
<meta name="msapplication-TileColor" content="#8b4513">
<meta name="theme-color" content="#8b4513">
